## Setup
Install the dependencies:

`pip install -r requirements.txt`

Make migrations:

`python manage.py makemigrations`

Migrate:

`python manage.py migrate`
## Run
`python manage.py runserver`

And navigate to `http://127.0.0.1:8000`.
